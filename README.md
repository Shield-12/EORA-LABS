# Signal & Systems — Langston Brown Portfolio

This is a static professional portfolio site for Langston Brown. It is designed to be simple to publish on GitHub Pages without npm, yarn, React, Next.js, or a build pipeline.

## What is included

- `index.html` — the complete portfolio page
- `styles.css` — Warm Walnut visual theme, responsive layout, and animations
- `script.js` — scroll reveal animations and opt-in sound button
- `.nojekyll` — helps GitHub Pages serve the files as-is
- `assets/docs/` — optional location for a public resume PDF or supporting documents

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder into the repository root.
3. Enable GitHub Pages for the repository using the main branch and root folder.
4. Wait for GitHub Pages to publish the site.

Because this is a static site, no package install is required.

## Privacy note

Certification PDFs were intentionally not included because they can expose verification codes and certificate IDs publicly. If you want public credential proof, use official verification links or a controlled document strategy instead of uploading every certificate PDF directly to a public repository.

## Optional resume download

If you want to add a downloadable resume later, place a public-safe PDF in:

```text
assets/docs/Langston_Brown_Resume.pdf
```

Then add a button in `index.html` linking to:

```text
assets/docs/Langston_Brown_Resume.pdf
```
